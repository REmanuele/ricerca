# 📊 C++ Sorting & Search Algorithms

A comprehensive collection of classic sorting algorithms implemented in **C++**. 
I created this repository during my university studies (Programming 2 course) to understand, implement, and analyze the performance of fundamental computer science algorithms.

---

## 🗂️ Implemented Algorithms

The repository is organized into directories, each containing the C++ implementation of a specific sorting method:

* **00. Ordinamento per scambio** (Exchange Sort)
* **01. Bubble Sort**
* **02. Selection Sort**
* **03. Insertion Sort**
* **04. Quick Sort**
* **05. Merge Sort**
* **06. Heap Sort**
* **07. Shell Sort**

---

## 🛠️ Built With

* **Language:** C++
* **Concepts covered:** Time complexity, Array manipulation, Recursion, Memory management.

---

## ⚙️ How to Compile and Run

To test any of these algorithms on your local machine, you need a C++ compiler (like GCC or Clang).

**1. Clone the repository:**
\`\`\`bash
git clone https://github.com/REmanuele/ricerca.git
cd ricerca
\`\`\`

**2. Navigate to a specific algorithm's folder:**
\`\`\`bash
cd 04.QuickSort
\`\`\`

**3. Compile and execute:**
\`\`\`bash
g++ main.cpp -o run_algorithm
./run_algorithm
\`\`\`
*(Note: Replace `main.cpp` with the actual name of the C++ file inside the folder).*
